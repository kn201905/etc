# 一般的な C# コーディング規約
* 変数名、関数名はキャメルケースで書き、「_」は用いない。
* 小文字始まり -> 変数名
* 大文字始まり -> クラス名、関数名、列挙型

MS公式の解説  
https://docs.microsoft.com/ja-jp/dotnet/csharp/programming-guide/inside-a-program/coding-conventions
